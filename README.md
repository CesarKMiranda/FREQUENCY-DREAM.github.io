# PROYECTO FINAL MODULO 1


## PROYECTO "FREQUENCY DREAM"

Buen dia sean bienvenidos al repositorio de mi proyecto final que he realizado para el primer módulo

Este documento sirve como guía para entender, configurar y ejecutar la aplicación principal

## VISION DEL PROYECTO

Este proyecto consiste en una página web estática simple, creada con el objetivo fundamental de aprender y aplicar los conceptos básicos de HTML, mi primer proyecto es una página web básica creada solo con HTML para presentar FREQUENCY DREAM, una escuela ficticia de diseño de sonido, mezcla y masterización la cual espero en un futuro lograr hacer, la página incluye un menú de navegación, promociones, productos con imágenes y botones (sin funcionalidad aún), una pequeña sección de prueba y un pie de página con redes sociales, al no usar CSS ni JavaScript, la página no es responsive, los botones no funcionan y solo los enlaces de redes sociales llevan a otras páginas
Este proyecto representa un momento muy importante, ya que es mi primer documento HTML escrito y estructurado por mi, si bien la funcionalidad es limitada, el enfoque principal ha sido asegurar una estructura semántica clara, siguiendo las mejores prácticas y consejos que mi sensei me otorgó en las clases.

## TECNOLOGIAS Y VERSION

#### ENFOQUE EN HTML5 BASICO

Se ha utilizado exclusivamente código HTML5 para la estructura, esto significa que los componentes se basan en etiquetas fundamentales como:

Estructura Base: !DOCTYPE html, html, head, body

Contenido: h1, h2, p, a, img

Listas: ul y li

Nota Importante: Este proyecto no incluye CSS ni JavaScript, ya que el objetivo era dominar primero la estructura basica de HTML

### Instrucciones de uso

Debido a que se trata de una página web estática y autocontenida, la configuración y el uso son muy sencillos

1. Requisitos

No se requiere software especializado más allá de un navegador web (Chrome, Firefox, Edge, Safari, etc.)

2. Ejecución

Sigue estos pasos para ver la página en acción:

- Localiza el Archivo: Asegúrate de que tienes el archivo principal del proyecto llamado index.html guardado en una ubicación conocida de tu ordenador

- Abre con el Navegador: Haz doble clic en el archivo index.html o bien accede a través del vinculo https:/

- El navegador abrirá automáticamente el documento y lo renderizará
